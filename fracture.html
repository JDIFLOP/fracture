กระดูกหักมีกี่ประเภท?

การแบ่งประเภทของกระดูกหักสามารถแบ่งได้เป็นหลายรูปแบบตามลักษณะและสาเหตุของการหักสามารถจำแนกได้เป็น 2 ประเภท ดังนี้

  • กระดูกหักแบบที่ไม่มีแผลเปิด : เป็นกระดูกหักที่เกิดขึ้นโดยที่ชิ้นส่วนของกระดูกยังคงอยู่ในเนื้อเยื่อ ไม่ได้ทะลุผิวหนังออกมาทำให้ไม่มีแผลเปิดที่ผิวหนัง กระดูกหักประเภทนี้มักจะพบได้บ่อย มีโอกาสฟื้นตัวได้ดีหากได้รับการดูแลรักษา อย่างเหมาะสม มักเกิดจากการหกล้ม อุบัติเหตุ หรือแรงกระแทก

  • กระดูกหักแบบที่มีแผลเปิด : เป็นกระดูกหักแบบที่มีกระดูกทะลุผิวหนังออกมา ทำให้มีแผลเปิด และกระดูกอาจสัมผัสกับภายนอก ซึ่งอาจเกิดจากการที่กระดูกถูกแรงกระแทกอย่างรุนแรงหรือถูกบิดอย่างแรง กระดูกหักแบบนี้มีความเสี่ยงสูงที่จะติดเชื้อ เนื่องจากกระดูกมีการสัมผัสกับเชื้อโรคภายนอก

กระดูกหักมีกี่แบบ?

การแบ่งประเภทของกระดูกหักสามารถแบ่งได้เป็นหลายรูปแบบตามลักษณะการหักของกระดูก โดยทั่วไปสามารถจำแนกได้เป็น 11 แบบ ดังนี้

  • กระดูกหักทั่วไป : กระดูกหักชนิดนี้เป็นการที่กระดูกหักออกเป็นสองชิ้น มักพบในอุบัติเหตุหรือการหกล้ม

  • กระดูกยุบตัว : กระดูกยุบตัวคือการที่กระดูกได้รับแรงกระแทกอย่างรุนแรงจนเกิดการยุบตัว โดยเฉพาะในกระดูกสันหลัง ซึ่งมักพบในผู้สูงอายุหรือผู้ที่มีภาวะกระดูกพรุน

  • กระดูกแตกเป็นชิ้นเล็ก ๆ : กระดูกหักชนิดนี้หมายถึงกระดูกที่หักออกเป็นชิ้นเล็ก ๆ มากกว่า 3 ชิ้นขึ้นไป มักเกิดจากอุบัติเหตุที่รุนแรง เช่น อุบัติเหตุรถยนต์ หรือการตกจากที่สูง

  • กระดูกหักแบบเกลียว : กระดูกหักแบบเกลียวเกิดจากการที่กระดูกถูกบิดหรืองอจนเกิดการหักเป็นลักษณะเกลียวหรือสกรู มักพบในการเล่นกีฬาหรืออุบัติเหตุที่มีแรงบิดสูง

  • กระดูกเดาะ : กระดูกเดาะหมายถึงกระดูกที่หักเพียงข้างเดียว ส่วนอีกด้านหนึ่งยังคงต่อกันอยู่ หรือเกิดการโค้งงอไปตามแรงที่กระแทก

  • ปุ่มกระดูกหัก : การหักชนิดนี้เกิดจากกระดูกถูกดึงหรือกระชากอย่างแรง ทำให้ส่วนของกระดูกที่ปุ่มหรือปมกระดูกหลุดออกไป มักพบในบริเวณหัวไหล่ หัวเข่า และข้อเท้า

  • กระดูกหักแนวขวาง : กระดูกหักในแนวขวางคือการหักของกระดูกที่เกิดในแนวขวางของกระดูก มักเกิดจากการกระแทกโดยตรงหรือแรงกระแทกที่เกิดขึ้นในแนวตั้งฉาก

  • กระดูกหักแบบเฉียง : กระดูกหักแบบนี้มักเกิดจากกระดูกที่หักในแนวเฉียงหรือลาดลง มักเกิดจากแรงกระแทกที่มาจากมุมเฉียง

  • กระดูกหักยุบเข้าหากัน : กระดูกหักชนิดนี้เกิดจากการที่กระดูกสองฝั่งถูกกดเข้าหากัน ทำให้กระดูกแตกและยุบลงทั้งสองด้าน มักพบในกรณีของอุบัติเหตุที่มีแรงกระแทกสูง

  • กระดูกหักจากความเครียด : เกิดจากการใช้งานกระดูกอย่างต่อเนื่องและยาวนาน จนทำให้กระดูกเกิดรอยร้าวเล็ก ๆ พบได้บ่อยในนักกีฬา

  • กระดูกหักจากโรคหรือความผิดปกติอื่น ๆ : เกิดจากภาวะหรือโรคที่ทำให้กระดูกอ่อนแอลง เช่น โรคกระดูกพรุน มะเร็งกระดูก หรือโรคที่ทำให้มวลกระดูกลดลง
